# BY INVEST — лендінг курсу від школи BY finance

Next.js 14 (App Router) + TypeScript + Tailwind CSS.

## Запуск

```bash
npm install
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000).

Прод-збірка:

```bash
npm run build
npm start
```

## Структура

```
app/
  layout.tsx            # обгортка (шрифти, мова uk)
  page.tsx              # головна сторінка (всі секції)
  globals.css           # глобальні стилі та компоненти Tailwind
  diagnostics/
    page.tsx            # окрема сторінка форми діагностики
components/
  Header.tsx            # фіксований хедер з навігацією
  Hero.tsx              # перший екран (ієрархія тексту як у шаблоні)
  Benefits.tsx          # 6 переваг навчання
  About.tsx             # «Про автора» — Юлія + виділені метрики
  Program.tsx           # 6 модулів з expand/collapse, акцент на 6-му
  Tariffs.tsx           # 3 тарифи: START / VIP / MAX, гарантія
  Bonuses.tsx           # ексклюзивний золотий блок «БОНУСИ»
  Cases.tsx             # карусель кейсів студентів
  Reviews.tsx           # 2 каруселі: Telegram-відгуки + YouTube ефіри
  Results.tsx           # нагороди школи
  CTASection.tsx        # фінальний CTA
  Footer.tsx            # футер
```

## Дизайн-нотатки

- Палітра: чорний `#0A0A0B` + золотий градієнт `#FBEDB6 → #E8B81B → #A07F1C`.
- Тарифи **VIP** і **MAX** виділені окремими класами з посиленою тінню та золотою рамкою.
- 6-й модуль у блоці «Програма» має акцентну рамку та мітку "Ексклюзивний модуль".
- Блок «БОНУСИ» — у золотій рамці на радіальному градієнті.
- Гарантія повернення коштів — окремий контрастний банер прямо під тарифами.
- Кнопки «Діагностика» по всьому сайту ведуть на окрему сторінку `/diagnostics`.
